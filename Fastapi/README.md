# FastAPI Learning Roadmap

## 1. Learn Python
**a. Python Installation:**
- Install Python 3.11 from the [official guide](https://www.python.org/downloads/release/python-3110/), download and set up locally.

**b. Learn Python Syntax:**
- If you are not familiar with Python, you can learn it from [W3Schools Python Tutorial](https://www.w3schools.com/python/) or refer to YouTube videos:
	- [Python Tutorial for Beginners (English)](https://www.youtube.com/results?search_query=python+tutorial+for+beginners+english)
	- [Python Tutorial for Beginners (Hindi)](https://www.youtube.com/results?search_query=python+tutorial+for+beginners+hindi)

## 2. Learn Basics of API
**a. Covers Topics Like:**
	- What is an API?
	- HTTP basics:
		- Methods: GET, POST, DELETE, PUT
		- Learn Status Codes
	- JSON Basics
	- URL Structure
	- Headers: Request and Response Headers
	- Authentication and Authorization

## 3. Learn MongoDB
Learn about databases. You can refer to YouTube videos:
	- [MongoDB Tutorial (English)](https://www.youtube.com/results?search_query=mongodb+tutorial+english)
	- [MongoDB Tutorial (Hindi)](https://www.youtube.com/results?search_query=mongodb+tutorial+hindi)

**a. Must DO topics:**
	- Setting up MongoDB
	- CRUD operations
	- Schema Designing

## 4. Basics of FastAPI
After covering the above topics, you can learn FastAPI from the following resources:
	- [FastAPI Documentation](https://fastapi.tiangolo.com/)
	- [FastAPI Tutorial (Hindi)](https://www.youtube.com/results?search_query=fastapi+tutorial+hindi)
	- [FastAPI Tutorial (English)](https://www.youtube.com/results?search_query=fastapi+tutorial+english)

## 5. Intermediate and Advanced FastAPI
After you have covered the basics, you are capable of creating CRUD-based APIs. For advanced and intermediate topics, refer to:
	- [FastAPI Advanced (Hindi)](https://www.youtube.com/results?search_query=fastapi+advanced+hindi)
	- [FastAPI Advanced (English)](https://www.youtube.com/results?search_query=fastapi+advanced+english)
